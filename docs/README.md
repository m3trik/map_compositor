### Installation:

###### 

To install:
Add the `map_compositor` folder to a directory on your python path, or
install via pip in a command line window using:
```
python -m pip install map-compositor
```

### Example use-case:
```
from map_compositor import Map_compositor_main
Map_compositor_main()
```
